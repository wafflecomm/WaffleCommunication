# 🧇 Waffle Communication (와플커뮤니케이션)

> **창의적인 플레이, 혁신적인 개발 — Funny Flow, Happy Connect**

와플커뮤니케이션은 캐주얼 게임, 에듀테크, 그리고 AI 음악 추천 서비스를 제공하며 기술과 재미의 경계를 허무는 혁신적인 소프트웨어를 개발합니다. 본 프로젝트는 와플커뮤니케이션의 프리미엄 브랜드 이미지를 전달하기 위해 최신 웹 기술과 디자인 트렌드를 결합한 공식 단일 페이지 애플리케이션(SPA)입니다.

---

## ✨ 핵심 기능 (Key Features)

### 🛸 Antigravity Hero (안티그래비티 히어로)
- **무중력 애니메이션:** 사용자의 시선을 사로잡는 마우스 반응형 부드러운 오브젝트 부유 효과.
- **Glassmorphism:** 현대적이고 투명한 느낌의 UI 레이아웃.

### 🎮 Premium Services
- **캐주얼 & 온라인 게임:** 소셜 엔터테인먼트를 위한 혁신적인 보드게임 경험.
- **교육용 앱 솔루션:** 재미와 배움을 효과적으로 융합한 에듀테크 서비스.
- **AI 음악 추천:** 사용자 무드에 맞춘 스마트 큐레이션 및 제작 서비스.

### 📚 Waffle Study
- **AI-Powered Workflow:** NotebookLM부터 Cloudflare 배포까지, AI 도구를 활용한 6단계 웹 서비스 제작 가이드 제공.

### 🤝 Global Partners
- Google, OpenAI, Anthropic(Claude), Ollama 등 글로벌 테크 리더들과의 파트너십 쇼케이스.

---

## 🛠 기술 스택 (Tech Stack)

- **Foundation:** Vanilla HTML5, CSS3, Modern JavaScript (ES6+)
- **Design:** CSS Variables, Grid/Flexbox Layout, Glassmorphism CSS
- **Interactivity:** Custom Scroll Reveal, Intersection Observer API
- **AI Ops:** Antigravity AI Configuration (Ollama & Gemini Hybrid)


## 🚀 시작하기 (Getting Started)

원격 서버나 빌드 과정 없이 정적 파일로 구성되어 있어 즉시 실행 가능합니다.

1. 본 저장소를 클론합니다.
   ```bash
   git clone https://github.com/zeons/WaffleComm.git
   ```
2. `index.html` 파일을 브라우저에서 엽니다. (Live Server 확장 프로그램 권장)
