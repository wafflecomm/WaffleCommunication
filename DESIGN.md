# Waffle Communication Design System

와플커뮤니케이션의 디자인 시스템은 **"미래지향적 프리미엄(Futuristic Premium)"**을 지향합니다. 다크 모드를 기반으로 한 글래스모피즘(Glassmorphism)과 역동적인 애니메이션 레이어를 통해 사용자에게 몰입감 있는 경험을 제공합니다.

---

## 🎨 1. Visual Theme & Atmosphere

WaffleComm은 심해의 차분함과 우주의 신비로움을 결합한 다크 팔레트를 사용합니다. 단순한 정적 웹사이트를 넘어, 사용자의 움직임에 반응하는 유기적인 레이아웃이 특징입니다.

- **심미적 특징:** 다크 모드, 반투명 유리 효과(Glassmorphism), 부드러운 그라데이션, 마우스 반응형 무중력 오브젝트.
- **핵심 감성:** 혁신, 창의성, 신뢰감, 유연함.

## 🎨 2. Color Palette

디자인 시스템의 색상은 시인성과 브랜드 정체성을 유지하기 위해 엄격하게 제어됩니다.

| 색상명 | 값 | 용도 |
| :--- | :--- | :--- |
| **Primary (Indigo)** | `#4F46E5` | 브랜드 핵심 컬러, 주요 버튼, 강조 텍스트, 아이콘 |
| **Primary Glow** | `rgba(79, 70, 229, 0.4)` | 버튼 호버 시 발생하는 광채 효과 |
| **Secondary (Amber)** | `#F59E0B` | 액센트 컬러, 비전 카드 타이틀, 강조 문구 |
| **Dark Background** | `#0F172A` | 섹션 배경, 푸터 베이스 |
| **Deepest Dark** | `#020617` | 메인 배경, 히어로 섹션 끝단 |
| **Main Text** | `#F1F5F9` | 헤드라인, 본문 텍스트 (높은 가독성) |
| **Muted Text** | `#94A3B8` | 서브 타이틀, 프로젝트 설명, 메타 데이터 |

## ✍️ 3. Typography

타이포그래피는 현대적인 세리프리스 폰트의 조합으로 전문성과 접근성을 동시에 확보합니다.

- **Headlines (Outfit):**
  - 기하학적이고 현대적인 인상의 폰트로 제목과 로고에 사용됩니다.
  - 적용: `h1`, `h2`, `h3`, `.logo-text`
  - 특징: `font-weight: 800`을 기본으로 하여 강한 존재감을 부여합니다.
- **Body & Logic (Noto Sans KR):**
  - 가독성이 뛰어난 산세리프 폰트로 본문과 리스트에 사용됩니다.
  - 적용: `body`, 일반 텍스트 설명.

## ✨ 4. Design Principles (Interaction)

### 🛸 Antigravity (무중력 애니메이션)
와플커뮤니케이션만의 시그니처 인터랙션입니다.
- **철학:** 기술은 딱딱하지 않고 부드럽게 흘러야 한다는 철학을 반영합니다.
- **구현:** 히어로 섹션 배경에서 부유하는 원형 오브젝트들이 마우스 좌표에 따라 마찰력과 관성을 가지고 반응합니다.

### 🧊 Glassmorphism (글래스모피즘)
모든 컴포넌트(카드, 폼, 내비게이션)는 유리 질감을 가집니다.
- **설정:** `rgba(255, 255, 255, 0.05)` 배경과 `backdrop-filter: blur(16px)`를 사용하여 심도를 표현합니다.
- **테두리:** `rgba(255, 255, 255, 0.1)`의 얇고 투명한 선으로 경계를 정의합니다.

### 🌊 Scroll Reveal
콘텐츠는 정적으로 고정되어 있지 않고, 사용자가 스크롤함에 따라 아래에서 위로 부드럽게 나타납니다.
- **Timing:** `0.8s`의 지속 시간과 `ease-out` 타이밍 함수를 사용하여 리드미컬하게 노출됩니다.

## 🧱 5. Component Standards

- **Buttons:**
  - **Primary:** 강한 그림자와 글로우 효과가 있는 인디고 버튼. 호버 시 상단으로 `3px` 이동하며 그림자가 확장됩니다.
  - **Secondary:** 외곽선만 있는 투명 버튼으로 배경을 투과하여 글래스모피즘을 강조합니다.
- **Cards:** 둥근 모서리(`30px`)와 부드러운 호버 회전 효과를 적용하여 입체감을 부여합니다.
- **Forms:** 입력 필드 포커스 시 브랜드 컬러(`--primary`)로 테두리가 강조되며 배경색이 미세하게 밝아집니다.
